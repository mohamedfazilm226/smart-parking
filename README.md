 Smart Parking System

## Description
This project is an IoT-based smart parking system that detects available parking slots using sensors and displays them on a website.

## Features
- Real-time parking slot detection
- Online slot booking
- Live status (Available / Occupied)

## Technologies Used
- HTML
- JavaScript
- Node.js
- IoT Sensors
- Cloud Database

## How to Run
1. Install Node.js
2. Run npm install
3. Run npm start
